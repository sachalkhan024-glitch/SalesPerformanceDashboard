# Sales Performance Dashboard

Analyzed 100K+ orders from the Brazilian Olist e-commerce platform to identify revenue trends, top product categories, and customer geography.

## Tools Used
- **Python** (pandas) — data cleaning, merging 4 datasets, monthly aggregation
- **Power BI** — interactive dashboard with KPIs, charts, and filters

## Key Findings
- Total revenue across 2016–2018 with clear monthly trends
- Top categories by sales volume and revenue contribution
- Customer distribution by state (SP, RJ, MG dominate)
- Order delivery time analysis and its impact on satisfaction
